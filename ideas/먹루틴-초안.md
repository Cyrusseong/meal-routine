# 먹루틴 (MealRoutine) — 서비스 기획 초안

> **출처:** overseas-service 크론잡 수집 (2026-02-21)
> **우선순위:** 1순위 — 커머스 연동 수익화 명확, 기술 장벽 낮음

---

## 한 줄 소개

한식 레시피 저장부터 주간 식단 짜기, 마켓컬리/쿠팡 장보기까지 한 번에 처리하는 식생활 관리 앱

---

## 벤치마크

| 항목 | 내용 |
|---|---|
| **원본 서비스** | Paprika (미국) |
| **원본 설명** | 레시피 수집 + 식단 계획 + 장보기 리스트를 하나의 앱으로 통합 |
| **원본 평가** | 미국 요리 커뮤니티(r/Cooking)에서 꾸준히 언급되는 유료 앱 |

---

## 한국에 없는 이유

만개의레시피·해먹남녀 같은 레시피 앱은 있지만, 식단 계획 → 재료 자동 취합 → 장보기 연동까지 이어지는 **통합 플로우가 없음**. 각 서비스가 사일로로 존재.

---

## 한국화 포인트

- **한식 특성 반영:** 한 끼에 반찬 여러 개 → "이번 주 반찬 세트" 단위 식단 플래닝 기능
- **커머스 연동:** 마켓컬리 / 쿠팡이츠 / 배민B마트 API 연동으로 재료 직접 주문
- **가구 분량 자동 계산:** 1인 가구 / 2인 가구 분량 자동 계산 (Paprika는 인분 조절만)
- **냉장고 파먹기 모드:** 남은 재료 입력 시 레시피 추천

---

## 타겟 유저

| 세그먼트 | 설명 |
|---|---|
| 2030 직장인 | 식비 절감에 관심 있는 층 |
| 3040 주부 | 아이 반찬 걱정하는 층 |
| 헬스 입문자 | 다이어트 식단 관리하는 층 |

---

## 수익 모델

| 구분 | 내용 |
|---|---|
| **프리미엄 구독** | 월 3,900원 — 무제한 레시피 저장 + 고급 영양소 분석 |
| **커머스 제휴** | 마켓컬리 등 클릭당 또는 구매당 수수료 |
| **확장 모델** | 식재료 큐레이션 구독 박스 (밀키트 버전) |

---

## 경쟁 우위

레시피 앱 → 커머스 플랫폼 연결 고리를 선점하면, 이후 직접 식재료 큐레이션 구독 박스(밀키트 버전)로 확장 가능.

---

## MVP 범위

> **목표:** 노코드 또는 소규모 팀으로 4~6주 내 검증 가능

1. **크롬 익스텐션** — 웹 레시피 원클릭 저장
2. **재료 자동 추출** — 저장된 레시피에서 재료 자동 파싱
3. **주간 식단 캘린더** — 끌어다 놓기 방식
4. **장보기 리스트** — 재료 취합 → 쿠팡 검색 링크 자동 생성 (API 없이도 가능)
