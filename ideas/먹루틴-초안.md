# 먹루틴 (MealRoutine) — 서비스 기획 초안

> **출처:** overseas-service 크론잡 수집 (2026-02-21)
> **우선순위:** 1순위 — 커머스 연동 수익화 명확, 기술 장벽 낮음
> **최종 수정:** 2026-02-21 — 글로벌 K-Food 건강 트렌드 + 반전 스토리텔링 + YouTube 순환구조 + 출처 신뢰 + UX 보완

## 한 줄 소개

다이어트·운동·건강 목표에 맞춰 직접 요리하거나 바로 주문하고, 모든 경로에서 커머스 수익이 발생하는 식생활 플랫폼

---

## 0. 왜 지금, 이 서비스가 필요한가 — "나도 그랬는데..." 반전 스토리

> **설계 원칙:** 각 이야기는 실제 뉴스·통계를 뼈대로 하되, 약간의 허구(인물·상황)를 더해 **"어? 이거 내 얘기인데?"** → **"헐, 이렇게까지?"** → **"그래서 어떻게 했는데?"**의 3단 반전으로 몰입을 유도한다. 모든 숫자에는 실제 출처를 표기한다.

### 에피소드 1: "월급은 올랐는데 왜 더 굶지?" — 직장인 지은 이야기

> 4년차 마케터 지은(30)은 올해 연봉이 200만원 올랐다. 그런데 이상하게 점심을 더 자주 굶는다.
>
> 삼성동 평균 점심값 **15,000원**. 한 달이면 **33만원**. 연봉이 올라도 점심값은 더 빠르게 오른다. "도시락 싸야지" — 작심삼일. 일요일에 뭘 해먹을지 30분 고민하다 넷플릭스를 켰다.
>
> **반전:** 어느 날 지은은 회사 동기가 매일 예쁜 도시락을 싸 오는 걸 봤다. "너 요리 잘하잖아?" 동기가 웃으며 답했다. **"아니, 나 요리 못 해. 앱이 이번 주 식단 짜주고, 재료 목록 만들어주고, 쿠팡 링크까지 다 해줘. 난 그냥 시키는 대로 장 봐서 따라 만들 뿐이야."**
>
> 지은이 절약한 금액: 월 **12만원**. 연봉 인상분보다 효과가 컸다.
>
> *출처: 직장인 평균 점심값 9,500원, 삼성동 15,000원 — [NHN페이코·스포츠경향 2025.07](https://sports.khan.co.kr/article/202507011042003), 도시락 월 10만원+ 절약 — 한국소비자원*

### 에피소드 2: "28살에 고혈압이라고요?" — 1인가구 민수 이야기

> IT 개발자 민수(28)는 건강검진에서 충격을 받았다. **고혈압 전단계**. 28살에.
>
> 민수의 하루: 아침 거르기 → 점심 회사 근처 국밥 → 저녁 배달 치킨 or 라면. 주 5일 배달, 나트륨 섭취량 **4,200mg** (WHO 권고량의 **2.1배**).
>
> "1인가구라 어쩔 수 없지 않나요?" 민수의 말에 의사가 답했다. **"어쩔 수 없는 게 아니라, 선택지가 없었던 겁니다."**
>
> **반전:** 민수는 냉장고에 남은 두부, 계란, 파 — 이 3가지로 만들 수 있는 레시피가 **47개**라는 걸 알게 됐다. 된장찌개, 두부 스크램블, 파전... 배달 대신 집밥을 선택한 날, 나트륨 섭취량은 **1,800mg**으로 절반 아래로 떨어졌다. 3개월 후 혈압은 정상으로 돌아왔다.
>
> *출처: 1인가구 배달 나트륨 2019년 146mg→2022년 301mg 2배 증가 — [서울경제 2024](https://m.sedaily.com/amparticle/13897433), 2030 세대 고혈압·당뇨 급증 — [질병관리청 국민건강영양조사 2023](https://www.kdca.go.kr/board/board.es?mid=a20501010000&bid=0015&list_no=726622&act=view), 1인가구 804만(전체 36.1%) — [통계청 2025](https://www.korea.kr/briefing/pressReleaseView.do?newsId=156734077)*

### 에피소드 3: "다이어트 앱 7개 깔아봤는데요" — 직장맘 수진 이야기

> 출산 후 12kg이 늘어난 수진(35). 다이어트 결심 후 앱을 깔았다. **눔, 밀리그램, 인아웃, 다이어트신, 필라이즈, 간단, YAZIO — 총 7개.**
>
> 기록은 열심히 했다. 사진 찍고, 칼로리 입력하고, 그래프도 예쁘게 나왔다. 그런데 3주차에 깨달았다. **"이 앱들은 내가 뭘 잘못 먹었는지만 알려주지, 내일 뭘 먹어야 하는지는 안 알려주잖아."**
>
> 레시피 찾으러 만개의레시피 → 재료 확인 → 메모장에 옮겨 적기 → 마켓컬리 검색 → 배민으로 대체... **앱 5개를 넘나드는 '앱 서핑'**을 하다 지쳤다.
>
> **반전:** 수진은 결국 앱을 다 지웠다. 그리고 1달 뒤 다시 시작했다. 이번엔 **하나의 앱**으로. "목표 칼로리 입력하면 이번 주 반찬 세트가 나오고, 장보기 리스트가 자동으로 만들어지고, 쿠팡 링크까지... 아, 이거 7개 앱에서 따로 하던 걸 하나로 해주는 거였어."
>
> **6개월 후 -8kg.** 비결은 의지가 아니었다. **동선이 짧아진 것**이었다.
>
> *출처: 밀리그램 120만 DL — [한국경제 2023](https://plus.hankyung.com/apps/newsinside.view?aid=202303084308i), 필라이즈 140만 회원 — [메디포뉴스 2025](https://www.medifonews.com/news/article.html?no=192221), 식단 앱 이탈 49.4%가 "앱을 잊어서" — [오픈서베이 2024](https://blog.opensurvey.co.kr/trendreport/ux-app-retention-2024/)*

### 에피소드 4: "일요일이 제일 싫어요" — 주부 현정 이야기

> 두 아이 엄마 현정(39)에게 일요일 저녁은 공포의 시간이다. **"이번 주 반찬 뭐 하지?"**
>
> 한식 한 끼 = 밥 + 국 + 반찬 3개. 월~금 점심·저녁 기준 반찬이 **30가지**. 매주 이걸 머릿속으로 계획하고, 만개의레시피에서 레시피 찾고, 메모장에 재료 옮기고, 컬리에서 하나씩 검색한다. **이 과정만 매주 1시간.**
>
> 남편이 물었다. "그거 왜 맨날 힘들어해?" 현정이 폭발했다. **"당신이 한번 해봐."**
>
> **반전:** 남편이 진짜 해봤다. 20분 만에 포기했다. 그날 밤 남편이 찾아낸 것: **주간 반찬 세트를 자동으로 짜주고, 재료를 합산해서 장보기 리스트를 만들어주는 앱.** "이거... 너 매주 일요일에 1시간씩 하던 거, 앱이 3분 만에 해줘."
>
> 현정이 돌려받은 시간: 매주 **1시간**, 한 달이면 **4시간**, 1년이면 **이틀**치 노동.

### 이 4개 에피소드의 공통 패턴

```
1단계: 공감           → "어, 나도 그래"  (일상적 고민 제시)
2단계: 악화/충격       → "헐, 이렇게까지?" (통계·사실로 심각성 인식)
3단계: 반전/해결       → "그래서 어떻게?"  (서비스가 해결한 장면)
4단계: 숫자로 증명     → "진짜네?"        (절약 금액, 건강 개선 등)
```

| 에피소드 | "나도 그래" 포인트 | 반전 | 숫자 증명 |
|---------|-------------------|------|----------|
| 지은 | 연봉 올랐는데 더 굶음 | 요리 못하는 동기의 비결 = 앱 | 월 12만원 절약 |
| 민수 | 28살 고혈압 | 냉장고 3가지 재료 → 47개 레시피 | 나트륨 절반, 3개월 후 정상 |
| 수진 | 다이어트 앱 7개 삭제 | 하나의 앱으로 6개월 -8kg | 동선 단축이 비결 |
| 현정 | 일요일 반찬 공포 | 남편이 20분 만에 포기 | 연 이틀치 노동 절약 |

**먹루틴은 이 4가지를 하나로 잇는다.**

## 핵심 구조

사용자의 진입점은 **건강 목표(다이어트, 운동, 건강 관리)**이고, 거기서 두 갈래 경로로 분기하며 **양쪽 모두 어필리에이트 수익**이 발생하는 구조:

```
                    ┌─────────────────────┐
                    │  사용자 건강 목표      │
                    │  다이어트 / 운동 / 건강 │
                    └──────────┬──────────┘
                               │
                    ┌──────────▼──────────┐
                    │  맞춤 식단 추천        │
                    │  (칼로리·영양소 기반)   │
                    │  📊 출처: 식약처 DB    │
                    └──────────┬──────────┘
                               │
              ┌────────────────┴────────────────┐
              ▼                                  ▼
    ┌─────────────────┐                ┌─────────────────┐
    │  경로 A: 직접 요리  │                │  경로 B: 바로 주문  │
    │  "오늘은 해먹자"    │                │  "오늘은 시키자"    │
    └────────┬────────┘                └────────┬────────┘
             │                                   │
    ┌────────▼────────┐                ┌────────▼────────┐
    │ 레시피 제공        │                │ 음식 추천          │
    │ (냉장고 파먹기 포함) │                │ (건강 목표 기반)    │
    │ 📎 출처 링크 표시   │                │ 📊 영양정보 표시    │
    └────────┬────────┘                └────────┬────────┘
             │                                   │
    ┌────────▼────────┐                ┌────────▼────────┐
    │ 식재료 장보기       │                │ 음식 주문          │
    │ 쿠팡 / 컬리 / B마트 │                │ 배민 / 쿠팡이츠     │
    └────────┬────────┘                └────────┬────────┘
             │                                   │
    ┌────────▼────────┐                ┌────────▼────────┐
    │ 💰 식재료 어필리에이트│                │ 💰 음식 어필리에이트 │
    │ (구매당 수수료)     │                │ (주문당 수수료)     │
    └─────────────────┘                └─────────────────┘
```

### 왜 이 구조인가?

현실에서 사용자는:
- 시간 있을 때 → 직접 요리 (월 10만원 절약 가능 — 한국소비자원)
- 바쁠 때 → 배달 주문 (평균 9,800원/끼 — NHN페이코)

**두 상황 모두에서 "건강한 선택"을 도와주고, 양쪽 모두에서 수익이 발생**하는 구조가 핵심.

### 핵심 차별점

> 위 서비스들은 모두 **"기록"** 또는 **"콘텐츠"**에서 멈춘다. 먹루틴은 **"기록이 아니라 실행"** — 건강 목표에 맞는 레시피를 추천하고, 재료를 장바구니에 담아주거나, 건강한 배달 메뉴를 바로 주문할 수 있게 연결한다.

## 한국에 없는 이유

각 영역의 강자가 따로 존재하지만, **사일로(silo)로 분리**되어 있음:

```
현재 한국 시장의 사일로 구조:

[만개의레시피]──── 레시피만 ────────────────────── 끝.
[밀리그램/눔] ──── 기록만 ─────────────────────── 끝.
[쿠팡/컬리] ───── 구매만 ─────────────────────── 끝.
[배민/쿠팡이츠] ── 배달만 ─────────────────────── 끝.

먹루틴이 잇는 플로우:

건강 목표 → 식단 추천 → 레시피 or 메뉴 → 장보기 or 주문 → 💰
```

**건강 목표 → 실행(요리/주문) → 커머스 수익**을 하나로 잇는 서비스가 없음.

## 데이터 신뢰 구조 — "출처 있는 건강 정보"

먹루틴이 다루는 레시피·영양·건강 정보는 반드시 **검증된 출처**에서 가져오고, 사용자에게 출처를 투명하게 표시하는 구조:

### 데이터 소스별 출처 체계

```
┌───────────────────────────────────────────────────────────┐
│  📊 영양 정보 — 공공 데이터 (출처 100% 명시)                  │
│  ├── 식약처 식품영양성분DB (data.go.kr/data/15127578)         │
│  │   → 26개 영양소, 무료 API, "출처: 식품의약품안전처" 자동 표기 │
│  ├── 전국통합식품영양성분 표준데이터 (data.go.kr/data/15100064) │
│  └── 식약처 맞춤 영양 진단 모델 (소스코드 공개)                 │
│                                                             │
│  📎 레시피 — 3계층 출처 관리                                  │
│  ├── Layer 1: 공공데이터포털 레시피 DB (식약처/농림수산식품)     │
│  │   → 출처: "공공데이터포털 / 식품의약품안전처" 자동 표기       │
│  ├── Layer 2: 사용자 저장 (크롬 익스텐션)                      │
│  │   → 원본 URL 자동 저장 → "출처: 만개의레시피 (원본 보기)" 링크│
│  │   → 재료·순서만 구조화, 원본 사진·서술 미복제               │
│  └── Layer 3: 사용자 직접 등록 (UGC)                          │
│      → 출처: "사용자 직접 등록" 표기                           │
│                                                             │
│  🏷️ 건강 콘텐츠 — 스토리 소재의 출처                          │
```

### 2. 과학이 증명한 한식의 건강 효과

#### 장 건강 (Gut Health)

- **스탠포드대학 연구 (2021):** 36명 대상 17주 무작위 임상시험. 김치 등 발효 식품 섭취 그룹이 **장내 미생물 다양성 증가** + **19개 염증 마커 감소** (인터루킨-6 포함). 고섬유 식단보다 효과 우수. *"식단의 간단한 변화가 건강한 성인의 미생물 군집을 재형성할 수 있다는 최초의 사례 중 하나"* — Justin Sonnenburg, PhD
  - *출처: [Stanford Medicine / Cell](https://med.stanford.edu/news/all-news/2021/07/fermented-food-diet-increases-microbiome-diversity-lowers-inflammation.html)*

- **김치 비만 억제 (2024):** BMI 23-30 성인 90명 대상 이중맹검 임상시험. 발효 김치가 **프로테오박테리아 감소 + 유익균 증가** 확인. 동물실험에서 김치가 *"장내 미생물, 스테로이드 호르몬, 담즙산, 대사 프로필을 현저히 변화"*
  - *출처: [PMC](https://pmc.ncbi.nlm.nih.gov/articles/PMC11435375/)*

#### 심혈관·대사 건강

- **메타분석 (2024):** 5개 중재연구(205명) + 4개 코호트(42,455명) 종합. 김치 섭취가 **공복 혈당 유의미 감소**, 중성지방·혈압과 역상관, 암·대사증후군 위험 감소, 정상 체중 유지와 연관
  - *출처: [PubMed](https://pubmed.ncbi.nlm.nih.gov/39545368/)*

- **한국 식단 vs 미국식 식단:** 한국 전통식이 비만 성인의 **분지쇄아미노산(BCAA) 유의미 감소** + 장내 미생물 조성 개선 (미국식 대비)
  - *출처: [PMC](https://pmc.ncbi.nlm.nih.gov/articles/PMC6835328/)*

#### 된장·고추장 — "한국의 3대 발효 식품" 과학적 근거

- **된장 항비만·항고혈압 (PLOS ONE, 2023):** 된장이 높은 염분에도 불구하고 지방 조직과 신장에서 레닌-안지오텐신 시스템을 활성화, **항비만 + 항고혈압 효과** 확인. *"된장의 소금은 단순 첨가 소금과 다르게 작용한다"* — 한국의 패러독스
  - *출처: [PMC](https://pmc.ncbi.nlm.nih.gov/articles/PMC10588895/)*

- **고추장 항고혈압 (PMC, 2024):** 약 10% 염분 함유에도 **레닌-안지오텐신-알도스테론 시스템 조절**을 통해 혈압 강하 효과. 대장염 동물 모델에서 표준 치료약(메살라민)과 유사한 항염 효과
  - *출처: [PMC](https://pmc.ncbi.nlm.nih.gov/articles/PMC11079108/), [PMC](https://pmc.ncbi.nlm.nih.gov/articles/PMC8150376/)*

#### 면역 체계

- **면역 강화 연구 (2025):** 12주간 김치 섭취 시 **항원제시세포(APC) 활성 강화** — 세균·바이러스 탐지 핵심 세포. *"유해한 과잉 반응을 방지하면서 면역 방어를 강화"*

### 5. 반찬(Banchan) 문화 = 세계 최고의 밀프렙 시스템

해외에서 **Meal Prep(식단 미리 준비)** 문화가 확산되면서, 한식 반찬 시스템이 **원조 밀프렙**으로 재조명:

```
반찬 시스템이 밀프렙에 최적인 이유:

✅ 대량 조리 전제   → 한 번에 만들어 일주일 활용
✅ 장기 보관 가능   → 발효·절임 반찬은 최대 6개월 냉장 보관
✅ 볶음 반찬       → 3~5일 신선도 유지
✅ 다양성 보장     → 한 끼 3~5종 반찬 = 영양소 자동 분배
✅ 칼로리 자연 조절 → 반찬 위주 식사 = 저칼로리·고영양
```

> **해외 적용 사례:**
> - 뉴욕 **Insa(인사)**: 계절별 로컬 식재료로 반찬 로테이션, 케일 등 현지 채소 활용
> - **Fit Men Cook**: 한식 반찬을 피트니스 밀프렙으로 소개
> - **US Foods**: 한식 반찬을 "weekly meal prep의 이상적인 구조"로 소개
> - **H Mart** (한국 슈퍼마켓 체인): 즉석 반찬·라면 매출 급증, 미국 전역 확장 중
> - **DoorDash**: 한식 점심 주문 2년 전 대비 **25% 증가** — 밀프렙 넘어 일상식 전환 신호

**HMR(가정간편식) 트렌드와의 시너지:**
불고기 키트, 냉동 잡채, 즉석 떡볶이 등 10분 이내 조리 가능한 한식 HMR이 해외 시장에서 급성장. 김치 토핑 생선, 비빔밥 피자, 불고기 타코 등 퓨전 한식도 확산.

1. **듀얼 어필리에이트:** 요리/주문 양쪽 모두에서 수익 → 사용자 행동에 관계없이 수익화
2. **"기록 → 실행" 전환:** 밀리그램(120만)·눔이 못 하는 "그래서 뭘 먹지?" 해결
3. **YouTube ↔ 앱 플라이휠:** 콘텐츠 = 마케팅 = 수익원 — 3가지를 동시에 해결
4. **반전 스토리텔링 바이럴:** "나도 그랬는데" 공감 → 쇼츠 바이럴 → 앱 유입
5. **출처 기반 신뢰:** 식약처 공공 데이터 + 뉴스 출처 → 건강 정보 신뢰도 확보
6. **극단적 UX 간소화:** 3탭·3초·3분 원칙 → 앱 이탈률 최소화 (경쟁 앱 73.4% 이탈 대비)
7. **냉장고 파먹기 → 장보기:** 절약 체감 → 재방문율 상승
8. **선점 효과:** 건강·레시피·커머스·YouTube를 잇는 포지션 → 밀키트 구독 박스 확장
9. **글로벌 K-Food 메가트렌드 탑승:** 미국 FDA 김치 권장 + TikTok 월 23억 조회 + UNESCO 2건 → "한식 = 건강식" 글로벌 인식 위에 서비스 구축, Phase 3에서 영문 확장으로 글로벌 TAM 극대화

## MVP 범위

> **목표:** 노코드 또는 소규모 팀으로 4~6주 내 검증 가능

### Phase 1: 경로 A (직접 요리) + YouTube 동시 시작

1. **YouTube 채널 개설** — 반전 스토리 쇼츠 5개 제작 (에피소드 1~4 + 냉장고 파먹기)
2. **크롬 익스텐션** — 웹 레시피 원클릭 저장 (원본 URL 출처 자동 보존)
3. **재료 자동 추출** — 저장된 레시피에서 재료 자동 파싱
4. **주간 식단 캘린더** — 끌어다 놓기 방식 + 칼로리/영양소 합산 표시 (식약처 DB 기반)
5. **냉장고 파먹기** — 보유 재료 입력 → 레시피 매칭
6. **장보기 리스트** — 재료 취합 → 쿠팡 검색 링크 자동 생성 (API 없이도 가능)
7. **YouTube 설명란 어필리에이트** — 영상별 재료 구매 쿠팡 파트너스 링크

### Phase 2: 경로 B (바로 주문) + YouTube 성장

8. **건강 목표 설정** — 다이어트/벌크업/유지 목표 + 칼로리 설정
9. **음식 추천** — 목표에 맞는 외식/배달 메뉴 추천 (영양 정보 출처 표시)
10. **배달 앱 연동** — 배민/쿠팡이츠 검색 링크 자동 생성
11. **YouTube 롱폼 시작** — 주간 식단 브이로그, 한 달 식비 챌린지

### Phase 3: 글로벌 확장 — K-Food 건강 플랫폼

12. **영문 UI/UX** — PWA 다국어 지원 (한국어/영어/일본어)
13. **영문 YouTube 채널** — "Korean Diet Challenge", "Why Koreans Stay Slim" 시리즈
14. **글로벌 커머스 연동** — Amazon Fresh (식재료) / Uber Eats·DoorDash (한식 배달)
15. **레시피 현지화** — 현지 마트에서 구할 수 있는 대체 재료 자동 추천 (예: 고추장 → Whole Foods 고추장 링크)
16. **과학 기반 콘텐츠** — Stanford/AHA 연구 인용한 "K-Diet Science" 섹션
17. **반찬 밀프렙 가이드** — "Korean Banchan = Ultimate Meal Prep" 콘텐츠 시리즈

> **Phase 3 근거:** 한국 식품 수출 $136억(2025), 미국 FDA 김치 권장(2026), TikTok K-Food 월 23억 조회 — 글로벌 수요가 이미 존재하지만 "따라할 수 있는 도구"가 없는 상태.
